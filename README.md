# Flask-Base

Flask-Base es un grupo de ficheros y directorios que forman la estructura de una aplicación base(boilerplate) en flask.

## Estructura del árbol del directorio

La organización de los directorios y ficheros esta compuesta de la siguiente forma:

```bash
flask_base/
    app/
        static/
        templates/
        __init__.py
        forms.py
        models.py
        routes.py
    .gitignore
    config.py
    name_app.py
    README.md
    requeriments.txt
```
